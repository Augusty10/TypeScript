# Why We Need TypeScript  
### Pros & Cons of JavaScript and Advantages of TypeScript

---

## 🚀 Introduction

JavaScript is one of the most widely used programming languages in the world. It powers the web, from simple websites to complex applications like Netflix, Instagram, and Google Docs. However, as applications grow larger and more complex, JavaScript starts showing limitations.

This is where **TypeScript** comes into the picture.

---

## 🤔 Why Do We Need TypeScript?

JavaScript is a **dynamically typed language**, meaning variables can change their type at runtime. While this provides flexibility, it also introduces bugs that are hard to detect during development.

### Example (JavaScript):

```js
let value = "10";
value = 10;
console.log(value + 5); // Unexpected behavior
